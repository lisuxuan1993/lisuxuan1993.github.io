把一个整形数组中重复的数字去掉 - 微软面试题

By Leo_璇哥

Feb 28 2013 更新日期:Sep 24 2014

文章目录

请把一个整形数组中重复的数字去掉。例如：
1, 2, 0, 2, -1, 999, 3, 999, 88
答案应该是：
1, 2， 0， -1， 999， 3， 88

思路：一次遍历数组找出绝对值最大的数，以便确定哈希表的大小。

第二次遍历数组，当遇到第一次出现的数，则放入应该放的位置（仍在原数组中），如果重复则不放入。
